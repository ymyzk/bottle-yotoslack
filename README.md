# bottle-yotoslack

## Requirements
* Python 2.7+ or 3.2+
* `requirements.txt`

## Configuration

### A: Use app.conf
* Copy `app.example.conf` to `app.conf`
* Edit `app.conf`

### B: Use environment variables
* Set environment variables
  * `YOTOSLACK_YO_CALLBACK_URL`
  * `YOTOSLACK_SLACK_CHANNEL`
  * `YOTOSLACK_SLACK_URL`
